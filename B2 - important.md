# B2 Telc Redemittel - Anki Flashcard Deck

Dieses Dokument enthält strukturierte Karteikarten (Flashcards) für Anki, um die wichtigsten B2-Konnektoren, Redemittel und grammatikalischen Muster so schnell wie möglich zu verinnerlichen. 
## 🎴 Rohtext für den Import (Tab-getrennt)
*Kopiere die folgenden Zeilen direkt in eine Textdatei für den Anki-Import:*
Die Ausbildung ist zwar anstrengend, ... (aber man lernt extrem viel.)	zwar ..., aber ... (Grammatik: Hauptsatz-Wortstellung nach "aber")
Ein Pflegeberuf erfordert nicht nur Fachwissen, ... (sondern auch viel Empathie.)	nicht nur ..., sondern auch ... (Grammatik: Aufzählung / Verstärkung)
Ich möchte sowohl theoretisch lernen, ... (als auch praktisch arbeiten.)	sowohl ..., als auch ... (Grammatik: Gleichwertige Aufzählung)
Je schneller ich Deutsch lerne, ... (desto besser kann ich kommunizieren.)	je ..., desto ... (Grammatik: je + Nebensatz [Verb am Ende], desto + Adjektiv + Verb im Hauptsatz)
Da ich mich sehr für Medizin interessiere, ... (habe ich mich für diese Ausbildung entschieden.)	da... (Grammatik: da = Nebensatz konjunktion [Verb ganz am Ende])
Ich helfe gerne Menschen, deshalb ... (arbeite ich im Krankenhaus.)	deshalb / darum / deswegen ... (Grammatik: Verb folgt sofort auf Position 2)
Ihre Anzeige hat mich sofort angesprochen, ... (da...)	Eröffnung (Brief) - Interesse bekunden
Als ich mich über ... informierte, bin ich auf ... (Ihre vielversprechende Anzeige gestoßen.)	Eröffnung (Brief) - Bezugnahme auf Anzeige
Leider wurden meine Erwartungen ... (nicht erfüllt, da...)	Beschwerde (Brief) - Enttäuschung ausdrücken
Sie können sich sicher vorstellen, wie ... (enttäuscht ich war, als...)	Beschwerde (Brief) - Nachdruck verleihen
Mir ist allerdings nicht ganz ... (klar, ob/wie...)	Nachfrage (Brief) - Unklarheiten ansprechen
Es wäre sehr nett, wenn Sie mir mehr Informationen über ... (zusenden könnten.)	Nachfrage (Brief) - Bitte um Infos
Der Text handelt von der Frage, ... (ob...)	Sprechen Teil 2 - Text zusammenfassen
Der Autor stellt ... (fest, dass...)	Sprechen Teil 2 - Kernaussage wiedergeben
Ich teile Ihre/deine Meinung ... (vollständig, denn...)	Sprechen Teil 2 - Zustimmen
Das klingt überzeugend, aber man muss auch ... (bedenken, dass...)	Sprechen Teil 2 - Teilweise zustimmen / Einwand bringen
Da muss ich dir/Ihnen leider ... (widersprechen.)	Sprechen Teil 2 - Höflich widersprechen
Was hältst du davon, wenn ... (wir...?)	Sprechen Teil 3 - Vorschlag machen (Planung)
Ein guter Vorschlag, aber wie ... (wäre es mit...?)	Sprechen Teil 3 - Gegenvorschlag machen (Planung)
Wir könnten uns darauf ... (einigen, dass...)	Sprechen Teil 3 - Einigung finden (Planung)
Wir müssen den Patienten ... (Hilfe leisten. [=helfen])	Nomen-Verb-Verbindung für "helfen"
Er zeigt großes ... (Interesse für den Beruf. [=sich interessieren])	Nomen-Verb-Verbindung für "sich interessieren"
Ich habe eine ... (Entscheidung getroffen. [=entscheiden])	Nomen-Verb-Verbindung für "entscheiden"
Wir müssen gemeinsam ... (eine Lösung finden. [=lösen])	Nomen-Verb-Verbindung für "lösen"
Der Patient muss morgen ... (geimpft werden. [Passiv mit Modalverb])	Passiv mit "müssen" (Struktur: Modalverb + ... + Partizip II + werden)
