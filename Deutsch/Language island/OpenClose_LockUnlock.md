# OPEN / Close

| Verb          |     | Action |
| ------------- | --- | ------ |
| öffnen        |     | open   |
| Auf \| machen |     | Open   |
|               |     |        |
| Schließen     |     | Close  |
| Zu \| machen  |     | Close  |

#### z.b.
ich mache Die tür an .
ich mache die Tür zu .
# Lock/Unlock

| Verb             |     | Action |
| ---------------- | --- | ------ |
| ab \| schließen  |     | Lock   |
| Auf \| schließen |     | Unlock |
#### Z.B.
- ch schließe die Tür ab.
- Ich schließe die Tür auf.
# State :
## auf/ein/an/zu + sein 
- **auf = up/open**
- **zu = shut/blocked**
- **an = active**
- **aus = dead/off**
- **ein = into system**
