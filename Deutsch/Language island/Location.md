### WO? (Where?) → Dativ
Location = no movement
- Das Buch liegt **auf dem Tisch**.
- Ich bin **in der Stadt**.

### WOHIN? (Where to?) → Akkusativ
Movement/change of location
- Ich lege das Buch **auf den Tisch**.
- Ich gehe **in die Stadt**.
# Pattern 
**Is it already there?** → Dativ  
**Is it going there?** → Akkusativ
# imp
> legen/stellen/setzen + hin == put/lye / place + there 
- ich stelle dieses Handy hin. : I am putting this phone there. 
> stellen/legen/setzen + zurück == put/lye/place + back 
- ich lege das Buch auf den Tisch zurück : im putting the buck back on the table .
