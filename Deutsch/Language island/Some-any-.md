| **irgendwohin**             | to somewhere             |
| **irgendwoher**             | from somewhere           |
| **irgendwann**              | sometime                 |
| **irgendwie**               | somehow                  |
| **irgendwelche**            | some / any (plural)      |
| **irgendein**               | some / any (singular)    |
| **irgendwoanders**          | somewhere else           |
### Common expressions (very important)
- **zum Schluss** = finally
- **am Schluss** = in the end
- **Schluss machen** = to end something
    - **Sie haben Schluss gemacht.**  
        → They broke up.
- **der Bewerbungsschluss** = application deadline
- **der Ladenschluss** = shop closing time
- **der Abschluss** ≠ **der Schluss**
    - **der Schluss** = the end
    - **der Abschluss** = graduation, qualification, completion, conclusion of a process
