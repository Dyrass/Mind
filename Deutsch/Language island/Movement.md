### Going 

| gehen  |     | to go     |
| ------ | --- |:--------- |
| fahren |     | to drive  |
| laufen |     | to run    |
| reisen |     | to travel |
#### Präposition
- **zu + Dativ** → destination (most important)
    - Ich gehe **zur Schule**
    - Ich gehe **zum Arzt**
- **in + Akk** → inside places
    - Ich gehe **in die Stadt**
    - Ich gehe **ins Kino**
- **nach** → countries/cities (no article)
    - Ich gehe **nach Berlin**
    - Ich gehe **nach Deutschland**

### Coming 

| Kommen           |     | come      |
| ---------------- | --- | --------- |
| an \| kommen     |     | arrive    |
| zurück \| kommen |     | come back |
#### Prep
- **zu + Dativ**
    - Kommst du **zu mir**?
	- Ich komme **zum Arzt**
- **aus + Dativ** = from inside
    - Ich komme **aus der Schule**
    - Ich komme **aus Deutschland**
- **von + Dativ** = from place/person
    - Ich komme **vom Arzt**
    - Ich komme **von meiner Freundin**
### Move in / out 
###### Ein | ziehen ––– eingezogen hat
Ich werde nach Deutschland einziehen.
###### Aus | ziehen ––– ausgezogen hat 
Ich ziehe aus der Wohnung  aus 

### Get in / Get out
###### Ein | steigen ––– eingestiegen _ist_
Ich bin an der Zug von Berlin eingestiegen 
###### Aus | steigen ––– ausgestiegen _ist_
Ich bin nicht zur Arbeit , weil ich am Auto schon ausgestiegen bin .
###### Um | steigen ––– umgestiegen _ist_
Bitte, wir sind zum Bahnhof erreicht worden. Alle sind gebeten, umzusteigen.
