# sein + aus/ein -> tells the state of that thing 
## Ein : on
- sein + Ein : tells the state that smth is on 
- Das handy ist ein
## Aus : off 
- Sein + Aus : tells the state that smth is off
- Das handy ist aus
---

| Verb            |     | Action       |                                     |
| --------------- | --- | ------------ | ----------------------------------- |
| An \| machen    |     | to turn on.  | Ich mache die Lampe an.             |
| ein \| schalten |     | to turn on.  | Sie schalt ihr Pc ein               |
|                 |     |              |                                     |
| Aus \| machen   |     | to turn off. | wir machen alle Lampen Zusammen aus |
| Aus \| schalten |     | to turn off. | hast du dieser Fan ausgeschaltet.   |
